categories-tags-setting

发表于 2023-04-03

首页 博客历程

所谓无规矩不成方圆，blog 才运作一年多的时间，但是由于不正确的分类导致自己在检索文章的时候出现了麻烦。虽说这种事情应该在开始就决定好，但是现在后知后觉意识到有效分类的必要性，也算是吃一蛰长一智了。因此特书此文，便于日后写文章的时候查阅相关信息...

在这里我们主要通过 categories 和 tags 进行文章分类，前期没有对这种细节过多关注，导致现在命名的 categories 和 tags 自己都认不出来了。

考虑到 categories 是 blog 检索的主键，tags 只是在 categories 基础上添加的一些信息，我决定将 categories 分成十组，分组如下：

  1. 记录生活
  2. 编程拓展
  3. 博客历程
  4. 图片分享
  5. 知识大陆
  6. 金曲共赏
  7. 校园学习
  8. 奇谈怪论
  9. 外文随笔
  10. 各抒己见

目前我能囊括的，能在博客上记录的应该只有上述内容，一口吃成胖子也不可能，日后慢慢完善也不着急，包括前面已经写好的 categories 和 tags 也有时间再来修改。
