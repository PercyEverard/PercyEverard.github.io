xlog/图床

发表于 2023-04-23
  • Percy Feng
首页

今天尝试接触了一下 xlog 博客平台，这是一个建立在区块链上的博客。在使用之前需要对区块链、代币有一定的认知，只是没想到第一次接触区块链是通过 blog！

不过对我来说，xlog 也好 hexo 也罢，都不过是记录 blog 的工具。我并没有不尊敬任何人的想法，只是可能我还不理解区块链去中心化的意义，不理解创作者推行 xlog 的苦心。是好是坏这一切都需要时间去检验。

我在 edge 浏览器上下载了一个名为 “微博图床” 的插件，可以拖拽本地或者网页上的图片形成图片链接。上周舍友向我提起 “图床” 这个词汇，乍一听我还以为又有什么新技术出现了，没想到只是图片转链接操作。如果不是写博客的话，我应该用不上 “图床” 这个词汇吧？

** 更新：** 微博图床插件的 GitHub 信息从 15 年更新到了 21 年，创作者能够在背后维护这款插件这么久真是让人敬佩！不过现在的微博图床貌似被 ban 了，已经无法从外部网站访问图片链接。上浏览器一找，发现网上博主哀嚎遍野，看来对于博主来说图床无法访问确实是一件让人困惑的事情。就算侥幸逃脱图片丢失的风险，还得重新把图片上传到图床并且手动修改网站相关的信息。如果斥巨资租服务器放图片又感觉得不偿失，果然还是白嫖的舒服！虽然我才刚接触到图床，但此时此刻我意识到选择一个好的图床很有必要😄！

我尝试了下面几个图床，效果都不错，都能够在我当前的 Markdown 上展示出来。

  1. 免费图床
  2. 路过图床，单张图片最高支持 10M
  3. 聚合图床
  4. sm.ms 图床，运行时间长，较为可靠
  5. 兰空图床，GitHub 高 star

当前我选择使用 sm.ms 图床，Markdown 格式或者 HTML 格式都可行。除此之外，还能够选择复制成 BBCode 格式。BBCode 是 Bulletin Board Code 的缩写，有译为「BB 代码」，属于轻量标记语言（Lightweight Markup Language）的一种，如字面上所显示的，它主要是使用在 BBS、论坛、Blog 等网络应用上。
