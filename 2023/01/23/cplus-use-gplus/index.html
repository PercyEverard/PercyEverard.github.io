要在 Ubuntu 编写 C 程序，那么需要有能编写代码的文本编辑和 C 编译器。 Ubuntu 环境里， 编译 C 语言的是 GCC ，编译 C 的是 G。执行下面的执指令安装编译 C 语言和 C++ 的环境：

sudo apt-get install gcc
sudo apt-get install g++
sudo apt-get install lsb-core lib32stdc++6

finishing installation check the version ensure everything is OK

g++ -v
gcc -v

# 2. create c++ file

first you should choose appropriate path to store the c++ file.

here are some linux instructs we usually see in working.

  • ls -->check the child folders in this Directory or catalogue
  • ls -a -->check the child folders and include the hidden folders
  • cd --> change directory
  • pwd --> show Present Working Directory
  • mkdir XXX --> make directory, means create a new folder
  • rm XXX --> remove files ANYONE but directory, means delete
  • rm -I XXX --> if you want to delete DIRectory, choose it!
  • touch XXX --> create any files you want!
  • gedit XXX --> create a temporary file e.g. TXT on the desktop
  • cat XXX --> print the text on terminal
  • code/nano/vi/vim --> VS/Nano/Vi/Vim text editor

maybe touch few -->

  • kill <ID#> --> kill appointed progress
  • ps -A --> print all progress now
  • ps ef | grep <#Keywords> looking for the progress ID you want, more conveniently

for more linux instructs you can visit here

Now we are going to create a c++ file use the instructs above.

cd Document/
cd cplusplusFile/
touch helloWorld.cpp

# 3. edit c++ file

it is unsuitable time for me to edit the c++ file with the help of vi/vim, i will learn this two editors next time so dont be depressed, i promise show it to you next time!

# 4. COMPILE

let's see the orders we use!

g++ helloWorld.cpp -o helloworld
./helloworld

Attention! don't forget -o, it means output. the "helloworld" after "-o" is the output file.
Finish all, we can see "Hello world!"print on the terminal! Congratulation!
