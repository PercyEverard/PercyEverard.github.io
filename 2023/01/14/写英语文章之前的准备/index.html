To improve my English skill, I decide to write English articles and keep a diary every day.But before I do that, I should make sure what I say is grammatically correct. And there is the reason I write this article.

# Incorporate common phrases

Writing an English article may be easy for me, even if it's bull shit. So I decide to incorporate some common phrases in my articles, you can interpret that as training. The following are phrases that I can refer to.

  1. 100 Common phrases in English
  2. 90 sentences in Real English
  3. Daily English conversation
  4. 873 Oral English Dialogue

# Check my grammar

I search for a suitable grammar checker for a long time and I find Grammarly is a useful tool. I just need to paste my English articles into that tool and I will know where the mistakes are. Originally I want to use the Web Version of Grammarly, but it seems to have some problems when I log in with my Google account. I have no way but to install this application on my computer. Even if I don't use scientific tools I can normally access the website and achieve my propose.
