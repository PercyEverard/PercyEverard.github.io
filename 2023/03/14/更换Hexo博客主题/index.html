更换Hexo博客主题

发表于 2023-03-14
  • Yume Shoka
首页

早就想给换掉博客主题了！之前用的 Next 也不是说不好用，就是有点简约过头了！我不推崇大道至简，我认为博客作为自己的门面怎么说也得增加点生气才行啊！于是...

Hexo 虽然好用，但是感觉上面的主题少之又少，物色了许久终于挑了几款心仪的！咳...！不开玩笑！发现几款还算入得了我眼的主题？结果想要 fork 过来看看效果如何，结果发现打开的主题网站要么就被 ban 了，要么就 disappear 了？这波啊！这波输在技术被卡脖子了！虽然我之前有学习过三件套的语法，但是要我自己写一个网站，还是有点力不从心的。

幸运的是，还有一个主题 Redefine 可以用，虽说还是简单，但不是灰白色感觉看起来舒服多了。接下来就开始迁移主题！

# 1. 打开你博客文件的 themes 文件夹

默认博客文件夹和主题文件夹是分开来的，这里只需要打开博客文件夹下的 themes 文件夹，并且右键 Git Bash 创建仓库，随后下载你要使用的主题。一般下载下来的主题文件由于文件夹层级关系的不对应，可能会让自己的博客无法正常显示出来。这个时候你需要将文件夹位置进行适当的调整。比如说 themes->Redefine->balabala files etc.

# 2. 更改_config.yml 文件

修改博客根目录下的 _config.yml 文件，注释掉原来的 theme 并新增一句 theme: xxx（注意冒号后面有空格）。themes 下的主题名称是啥你 xxx 就写啥，不然也是会报错的。

# 3. Deploy！

弱者才预览，强者直接上手！完成三部曲，实现博客的部署。

hexo c && hexo g && hexo d

至此完成！...
