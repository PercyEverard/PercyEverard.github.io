# 5. 7 月 31 日修正 AMENDMENT

这里记载的都是凭借着我的记忆和理解所写的（没有依赖互联网），如有不当或者不妥之处，恕我懒得修改！😜😜😜

  1. 按下 i 或者 a 进入 insert 模式
  2. 按下 esc 进入普通模式
  3. 键入 shift+: 进入 command 模式
  4. 键入 ctrl+v 进入可视化 visual 模式
  5. 普通模式下键入 v 或者 ctrl+v 表示选择 choose
  6. 选择 choose 后键入 yy 还是 y 可以复制 copy
  7. 找到空行然后键入 p 将复制的粘贴 paste
  8. 键入 ctrl+r 表示 redo 重新进行上一条指令？
  9. command 模式下键入 :u 表示返回上一条执行的指令
  10. command 模式下 :dd 或者 :ndd 表示删除 n 行 delete
  11. :q 不保存 quit
  12. :w 保存
  13. :wq 保存然后 quit
  14. :!q 强制 quit
  15. :!w 强制 write
