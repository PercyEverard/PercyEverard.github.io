Ubuntu 下安装 Git

发表于 2023-03-10
  • Percy Feng
首页 知识大陆

今天转战 Ubuntu 的 git，虽然是和 windows 上安装 Git 的步骤大同小异。不过还是记录一下，下次再次在 Linux 环境下安装 Git 就方便多了！

这次想要在 Ubuntu 上安装 Git 的契机是因为找到一份在 Linux 环境下的 Qt 工程文件，如果在 Windows 上的 Qt 运行的话会显示缺少 dll 动态库。当然在 Windows 上是可以解决这个问题的，可以通过 MinGW 还有 Cmake 模拟 g++ 编译器对文件进行编译。但对于 Linux 指令知之甚少的我来说，这或许是一次熟悉 Linux 指令的一次机会...

# 在 Ubuntu 上安装 Git

sudo apt-get install git

# 完成 Git 的用户配置

git config --global user.name "your github's name"
git config --global user.gmail "your email binding to the github"

# 创建验证所需的 ssh 公钥

在本地创建一个 ssh 公钥，因为 Git 是通过 ssh 的方式来访问资源库的。

ssh-keygen -C "your email binding to the github" -trsa

随后一路回车，会在用户目录下创建对应的密钥文件，我这里的用户目录是 wenfeng。

# 在 GitHub 上传公钥

创建完公钥后，需要上传。使用命令 cd /.ssh** 进入 /.ssh 文件夹，输入 gedit id_rsa.pub** 打开 id_rsa.pub 文件，复制其中所有内容。接着访问 github、gitlab 或者其他 git, 点击 SSH 公钥，标题栏可以随意输入，公钥栏把你刚才复制的内容粘贴进去就 OK 了。这样你的电脑就可以和服务器仓库建立链接了。

# 下载 GitHub 文件

git init
git clone https://github.com/bottod/CloudMusic.git

完成下载！
