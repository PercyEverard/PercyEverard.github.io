# 一、视频理解的内容

机器视觉作为一种研究使机器视觉能力的科学，是人工智能领域一个重要的研究内容，操作对象主要包括图像和视频两类。视频的本质是图像，简单来说可以看成是连续的序列图像，但是两者之间又有差别，如，视频中存在时序信息，即各序列帧之间存在顺序关系，如何利用视频中的时序信息是视频理解研究中的关键问题；此外，视频还是多模态的，除时序信息外还包括声音、文字等内容，如图 1.1 所示。有些应用场景中需要对视频中的声音、图像、文字进行综合考虑才能提取直观、立体的认知信息，即视频理解。

从底层理解，视频理解有几个问题：

  1. 分类、识别问题，是指对视频的内容进行分类，识别出视频中的目标
  2. 检测问题，是指不仅需要识别出视频中的物体，还需要定位出物体的位置、大小，颜色等属性
  3. 分割问题，主要是处理语义级别的分割问题，需要精确到像素级别，即要为图像中的每个像素分配一个标签，也可以看成是对图像中每个像素进行分类

基于底层的基础内容，视频理解又可以衍生出一些更深层次的技术，包括视频目标跟踪、主体描述、语义理解、事件检测、行为检测、文本视频相互生成等。更广泛的应用层面，视频理解需要解决的问题主要包括主体（who），指视频中包含的主体；时间（when），事件发生的时间；地点（where），事件发生的场景；事件（what），发生了什么事情。视频理解的研究，通常需要首先解决的问题是定义一个标准化的标签体系，该标签体系应该覆盖视频理解中的主体、地点、时间、事件等概念。建立标签体系有一定的难度主要体现在：

  1. 每天都会产生成千上万的视频，视频数据量较大，难以对所有样本标注标签
  2. 视频内容发生的场景较为复杂，通常视频中场景无法预测
  3. 视频中不同时段内，主体、事件、行为等都可能变化，且这种变化具有不可预测性，如行为的边界条件不易定义

# 二、视频理解的意义

人类从外界获取的信息中有 70% 到 80% 是来自于视觉信息，其中在视频上花费的时间是图片上的 2.6 倍。特别是随着科技的进步和发展，越来越多的人有意或无意地成为视频的制作者、传递者和消费者。在中国，随着抖音、快手等新兴社交软件或新传媒技术的发展，视频的爆炸式增长时代正在到来。根据统计，2017 年互联网上的流量 70% 来自于视频，中国网络视频用户规模达到 5.65 亿，这一数字在 2019 年达到了 7.59 亿，占网民整体的 88.8%，而且未来也呈现出稳定增长的态势。在国际互联网上视频数据也呈现出较快的增长趋势，以 Youtube 为例，全世界每秒传递的视频也达几千个小时。一方面，视频数据是人类获取外界信息的重要来源，另一方面，大量的视频数据单纯的依靠人工进行数据标注、内容分类、信息提取等处理不太现实，不仅费时费力、效率低下，而且还会引入人为造成的误差。对视频理解进行研究具有广泛的应用需求。

人工智能自 20 世纪 50 年代诞生以来，经过 1956-1976、1976-2006、2006 至今三次发展浪潮后，已经取得了较大进步，极大地改变了人们的生活生产方式。算法、算力和数据作为引领人工智能发展的三驾马车在人工智能的发展中起着至关重要的作用，且已经得到较为长足的发展，对人工智能起到了推动作用，提供了发展契机。算法是指结构、原理、逻辑等方面的突破，深度学习本质上就是一种算法。深度学习的训练和推理通常涉及到大量数据的并行计算，这就需要很大的算力，算力的提升一方面是取决于系统逻辑结构的改进，但更重要的是硬件结构的改进，如 GPU、FPGA、ASIC 以及其他专门用来处理神经网络的 AI 芯片等都具有并行计算能力。随着计算机的应用、互联网技术的发展，人们获取的数据规模越来越大。人工智能的发展水平已经能够为视频理解提供技术基础支持，对视频进行理解也是技术发展的趋势。

视频理解在深度学习技术的推动下，和知识表示、逻辑推理、语言处理等其他人工智能技术的交互呈现交互推进的态势。虽然视频理解技术得到较大发展，但是仍然有很多科学问题需要解决，仍然面临很多挑战，具有很大的研究空间。

特别是近些年，视频数据的大爆发，使得视频理解的研究成为了机器学习领域一个重要的课题。

# 三、视频理解的基本技术

广义上看，视频理解的技术范围很广，只要能从视频中通过计算机提取出信息的技术都可以视为视频理解技术，如视频中车辆识别技术、视频中人脸检测技术、视频中目标检索技术等。这里介绍的视频理解技术，主要是指针对视频序列的内容进行理解的技术，大致归为经典方法和逐帧融合的方法。

深度学习算法成熟之前的视频理解算法都可以划分为经典算法，其中具有代表的是 Wang 等人提出的 DT（dense trajectories）和 iDT（improved dense trajectories）方法。DT 算法采用 SVM 支持向量机分类器进行分类，iDT 是基于 DT 算法的改进。因为视频理解过程中处理的数据量较大，所以这类算法通常较慢，具体实现这里就不在详细介绍，感兴趣的同学可以课下自己研究，这也是神经网络出现后该类算法逐渐退出人们视线的原因之一。

视频通常是包含文本、语音、图像的多模态数据结构。在基于神经网络的视频理解中，融合处理是一种常用的特征提取方法。融合处理是指多个模态或多帧之间的信息进行融合。融合的方式大致可以分为两种：早期融合（early fusion）和晚期融合（late fusion）。这里的多模特融合是指视频中文本、语音、图像等多种模态自己的融合。

https://s2.loli.net/2023/05/22/Y3LtV1MSyecpPuO.png
"> <br> <div style="color:orange; border-bottom: 1px solid #d9d9d9; display: inline-block; color: #999; padding: 2px;"> 图 1.2 多模态不同融合方式 </div> </center>

如图 1.2 所示，早期融合是指多个模态特征在分类任务之前进行融合，然后将融合特征输入分类器进行分类，晚期融合是指多模态特征各自进行分类，然后将各自的分类结果进行融合。

当前对视频理解的研究还存在一定的局限性，在视觉领域，对于视频理解的研究集中在视频图像帧的模态。本课程实操部分主要讲解的内容主要包含视频理解中的多目标跟踪技术，视频目标跟踪主要是基于视频的视觉信息，检测并跟踪视频场景中的目标物体，因此在跟踪过程中并没有过多的考虑文本、语音等模态的信息，下面将对视频视觉信息的融合进行介绍。

如果单纯的把视频看出是一系列图像的集合，每帧图像可以单独提取特征，在融合它们的深度特征。如果将视频划分成固定长度的若干片段，就可以设计多种融合方法，较为常见的融合方法包括：单帧融合、前期融合、后期融合。

<center> <img style="border-radius: 0.3125em; box-shadow: 0 2px 4px 0 rgba(34,36,38,.12),0 2px 10px 0 rgba(34,36,38,.08);" src="
https://s2.loli.net/2023/05/22/HDeTWJNbU3vqBE4.png
"> <br> <div style="color:orange; border-bottom: 1px solid #d9d9d9; display: inline-block; color: #999; padding: 2px;"> 图 1.3 视频帧融合方式 </div> </center>

如图 1.3，单帧融合方式是每次只对单帧图像进行前馈网络处理；早期融合是在特征提取过程中直接设计网络结果对 5 帧图像同时提取特征（如将卷积核有 11113 改为 11113*5）；晚期融合将相距 15 帧的两帧图像分别输入前馈神经网络，然后对深度特征进行融合。一般来说早期融合的效果不如晚期融合的特征，主要原因可能是这种方式丢弃了时序信息，直接从图像中提取物体的运动信息难度较大。

视频理解的技术和方法发展较快，在选择方法时候要根据实际需要进行选择。同学们通过这一章的学习，只需要对视频理解有一定的了解就可以了，并不影响后续内容的学习，在学习后续章节内容时，也不要太局限于本章所提到的方法。

# 四、视频内容理解的应用场景

视频理解有着广泛的应用基础和应用前景，如可以用在人机交互上，比较传统的交互方式如鼠标或键盘，基于视觉理解的交互方式系统可以通过识别手势或人体姿态甚至人脸表情等做出相应的反应；在机器人的开发中可以用于基于视频的地图绘制、自助导航等。

在视频智能监控上，可以从实时监控场景的视频中提取关键信息，对视频中可疑的活动、行为或事件进行检测和识别，当发生异常行为时能够及时地自动预警或采取其他预防措施，如在自动取款机旁视频理解出有人员携带攻击性武器打斗时可以自动报警，在地铁、高铁站台视频理解出有人员晕倒、坠轨等危险行为时自动响起警报或关停列车等。

在智能交通中，可以通过对监控视频中车辆进行检测和行为识别，对车辆车型识别以及车流量统计，判断是否存在逆行、压线、超载、闯红灯等违反交通规则的行为。

在辅助驾驶中，可以对监控视频内容进行提取和理解，根据路况的识别结果控制车速、路牌标志的识别结果提醒驾驶人做出相应的规范操作、驾驶人面部表情或行为的识别结果预防疲劳驾驶等。

在军事上，可以用于基于视频的入侵检测、基于视觉的导弹导航或敌我识别、目标搜索、夜视红外影响的增强或信息隐藏等。

此外，视频理解还可以用在智能推荐、医疗机器人、智能家居等方面；互联网也是视频理解的重要应用场景，如在线视频、短视频、直播领域的应用，视频广告的精准推荐等，该领域已经成为继智能安防之后，另一个驱动万亿市场的应用。

视频理解的很多应用场景都需要对运动的目标进行跟踪并提取跟踪目标的属性信息，然后在此基础上进行更高层次的语义分析，最终得到视频内容信息的理解。本课程实操部分将实现一个基于视频的目标车辆的跟踪与属性识别系统，下一章将对视频理解中的目标跟踪进行将要介绍。在此约定，本课程接下来的篇章中，如果没有特殊说明，所提到的目标跟踪均指基于视频的目标跟踪。
