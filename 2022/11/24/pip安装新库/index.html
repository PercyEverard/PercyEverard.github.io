书接上回，研究一下在 visual code 上如何通过 pip 安装库函数，这次要安装的库函数是 skimage

# pip 安装 skimage 库

研究如何在 visual code 上如何通过 pip 安装库函数

# 找到 Python 的路径

我的是 *“D:\Python\Scripts”*

visual code 或者 cmd 命令台执行

cd D:\Python\Scripts

新发现，我的 Python 是全局的，所以直接打开 cmd 便可以执行 pip 指令，这样就免去了切换到指定目录下的过程

然后输入 pip install skimage

# pip 版本要更新

没办法，照着提示更新

# 更新完下载仍然报错

D:\Python\Scripts>.\pip install skimage

原来是库函数命名的问题，为什么我要的是

from skimage import io, data

但是却给了我个 “scikit-image” 呢？

不知道，更改名字后执行

D:\Python\Scripts>pip install scikit-image

成功！
