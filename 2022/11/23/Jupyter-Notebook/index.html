# Jupyter Notebook 常见快捷键操作

首先要知道 Jupyter 有三种 cell 类型

  1. Code
  2. Markdown
  3. Raw NBConvert（普通文本，运行不会输出结果）

两种模式

  1. 编辑模式（Enter）
  2. 命令模式（Esc）

# 1. 两种模式都可以使用的快捷键

  • Shift+Enter 执行本单元代码，并且跳转到下一单元
  • Ctrl+Enter 执行本单元代码，留在本单元

# 2. 命令模式

  • Y：cell 切换到 Code 模式
  • M：cell 切换到 Markdown 模式
  • A：在当前 cell 的上面添加 cell
  • B：在当前 cell 的下面添加 cell
  • 双击 D：删除当前 cell
  • Z：回退
  • Ctrl+Shift + 减号：分隔 cell，在光标处
  • L：为当前 cell 加上行号

# 3. 编辑模式：按 Enter 或鼠标单击代码块内部进入

  • Ctrl + 鼠标单击：多光标操作
  • Ctrl+Z：回退
  • Tab 键：代码补全
  • Ctrl：注释多行代码
